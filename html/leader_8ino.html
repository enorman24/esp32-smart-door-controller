Detailed Description

Leader ESP32 smart door controller.

Author
Elliot Norman
Melih Boyun
Date
2025-12-02

This file implements the "leader" ESP32 node for a smart door system. It:

  • Manages the LCD user interface.
  • Receives password input over serial and BLE.
  • Verifies passwords and commands the follower ESP32 over I2C.
  • Uses a hardware timer and FreeRTOS queue to automatically re-lock the door after a fixed interval.

◆ onDoorTimerExpire()

void IRAM_ATTR onDoorTimerExpire ( )

Hardware timer ISR for automatic door locking.

This ISR is triggered when the door auto-lock timer expires. It:

  • Creates a MessageForQueue with source S_LOCK_TIMER.
  • Enqueues the message using xQueueSendFromISR so the manager task can safely perform the lock operation in task context.
Note
Runs in interrupt context; work is intentionally minimal and deferred to taskSystemManager via the queue.

◆ passwd_correct()

void passwd_correct ( )

Handle correct password entry.

This function:

  • Updates the LCD to show the unlocked state.
  • Sends an "unlock" command (1) to the follower ESP32 over I2C.
  • Starts the hardware timer that will re-lock the door after UNLOCK_TIME_US microseconds.

◆ taskSerialInput()

void taskSerialInput ( void * pvParameters)

FreeRTOS task to read password input from the serial interface.

The task:

  • Accumulates characters into a temporary String until a newline is seen.
  • Trims whitespace and ignores empty lines.
  • Converts the string into a MessageForQueue with source S_SERIAL.
  • Sends the message to xSystemQueue for verification.
Parameters
pvParametersUnused task parameter (required by FreeRTOS signature).

◆ taskSystemManager()

void taskSystemManager ( void * pvParameters)

Main system manager task that consumes messages from the queue.

Responsibilities:

  • Initially locks the door at startup.
  • Blocks on xQueueReceive waiting for messages from:
    • taskSerialInput (S_SERIAL)
    • MyCallbacks::onWrite (S_BLE)
    • onDoorTimerExpire (S_LOCK_TIMER)
  • On S_LOCK_TIMER: locks the door and stops the auto-lock timer.
  • On S_SERIAL or S_BLE: compares the received password text to CORRECT_PASSWORD and calls passwd_correct or passwd_incorrect accordingly.
Parameters
pvParametersUnused task parameter (required by FreeRTOS signature).
