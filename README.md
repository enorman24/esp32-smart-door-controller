# ESP32 Access Control System

A dual-ESP32 embedded access control system featuring a password protected
door, alarm system, and motion controlled motor to open the door. Developed in VS Code using the
Arduino framework.

## Features
- Motor controlled door
- Passwork unlock via serial monitor or bluetooth low energy connection
- Alarm system if motion is detected while armed
- Motion detector to open door or sound alarm
